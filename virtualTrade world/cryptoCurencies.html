About this project

This project is a web application that provides real-time information on cryptocurrency prices.
Users can search for a specific cryptocurrency by symbol and view detailed information about each coin.
The application allows users to mark their favorite coins, up to a maximum of 5, and view them in a separate section with real-time reports on their prices.
The application also uses caching to reduce API calls and improve performance.
For this project, I used HTML, CSS, and jQuery to create a user-friendly design and present the information in the fastest and most convenient way possible.
